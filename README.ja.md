# AI Usage Monitor

Claude と Codex のWeb使用量ページを読み取り、デスクトップに常時表示する Windows ウィジェットです。

## できること

- Claude と Codex の使用状況を、常時最前面のウィジェットでまとめて確認できます
- 5時間枠と週間枠の両方を表示します
- 残量表示と使用量表示を、サービスごとに切り替えられます
- Codex のみ・Claude のみ・両方、必要なサービスだけ表示できます
- 横並びと縦並びをワンクリックで切り替えできます
- リセット表示は「あと ○時間 ○分」と「リセット時刻」を選択でき、5時間・週間それぞれ独立に設定できます
- 残量が少なくなると、黄→赤と進行バーの色で警告します
- リセット直前は **ブースト** で高頻度更新に切り替えられます

## 使い方

1. Releases から `AiUsageWebView2.zip` をダウンロードし、任意の場所に解凍します
2. `AiUsageWebView2.exe` をダブルクリックして起動します
3. 初回のみ、各カードの **ログイン** ボタンを押して Claude / Codex にログインしてください。一度ログインすれば、次回以降は自動取得されます

> WebView2 ランタイムが必要です（Windows 11 と最近の Windows 10 にはプリインストール済み）

## 画面

### 両サービス・横並び（デフォルト）

![基本表示](docs/01-overview.png)

Claude と Codex の 5時間枠・週間枠を、パーセンテージとプログレスバー、リセット時刻とともに表示します。右側のサイドレールに全体操作用のボタンが並んでいます。

### 片方だけ表示

![片サービス表示](docs/02-single-service.png)

片方しか使わない場合は、設定の「一般」から無効化できます。1枚カードのコンパクトな表示に切り替わります。

### 縦並びレイアウト

![縦並び](docs/03-layout-vertical.png)

2枚のカードを縦に積みます。デスクトップの端に縦長で置きたいときに便利です。サイドレールのレイアウトボタン、または設定の「レイアウト」から切り替えできます。

### 表示モードとリセット形式

![表示モード切替](docs/04-display-modes.png)

各サービスごとに **残量 / 使用量** を切り替えできます。リセット時刻は「残り時間」と「リセット時刻」を、5時間枠と週間枠それぞれ独立に選択できます。Claude / Codex の元ページで表記が違っていても、アプリ内部で日時に変換してから表示するため、形式が揃います。

## ボタン

| ボタン | 機能 |
|--------|------|
| ⚡ | ブースト（1分間隔の高頻度更新、30分間） |
| ↻ | 今すぐ手動更新 |
| × | 閉じる |
| 📌 | 最前面に固定 / 解除 |
| ⇆ | 横並び / 縦並び 切り替え |
| ⚙ | 設定を開く |

カーソルを合わせるとツールチップが出ます。

## 設定

右側サイドレールの ⚙ アイコンから開けます。

### 一般
| 項目 | 初期値 | 補足 |
|------|--------|------|
| Language | 日本語 | 日本語 / English |
| 最前面に固定 | 無効 | |
| Codex | 有効 | サービスの表示 ON/OFF |
| Claude | 有効 | サービスの表示 ON/OFF |

### レイアウト
| 項目 | 初期値 | 補足 |
|------|--------|------|
| 配置 | 横 | 横並び / 縦並び |
| Codex 数値 | 残量 | 残量 % / 使用量 % |
| Claude 数値 | 残量 | 残量 % / 使用量 % |
| 5時間リセット表示 | 残り時間 | 残り時間 / リセット時刻 |
| 週リセット表示 | リセット時刻 | 残り時間 / リセット時刻 |

### 更新
| 項目 | 初期値 |
|------|--------|
| 通常更新間隔（分） | 15 |
| ブースト時間（分） | 30 |
| ブースト更新間隔（分） | 1 |

### 閾値
| 項目 | 初期値 | 補足 |
|------|--------|------|
| 黄色になる残量（%） | 50 | 残量がこの値以下になるとバーが黄色 |
| 赤になる残量（%） | 30 | 残量がこの値以下になるとバーが赤 |

## 仕組み

非表示の WebView2 を 2つ、Claude / Codex の使用量ページに開きっぱなしにし、レンダリングされたテキストを読み取って、パーセンテージとリセット時刻をパースしています。表示はダークテーマで自前描画。ログインセッションは `%LOCALAPPDATA%\AiUsageWebView2\` 配下の WebView2 ユーザーデータフォルダに保存され、認証情報は外部に送信しません。

## ソースからビルド

```powershell
.\build.ps1
```

Windows と .NET Framework 4 が必要です（`build.ps1` 内に csc.exe のパスがハードコードされています）。WebView2 の NuGet パッケージは初回ビルド時に自動取得されます。
