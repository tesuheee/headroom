# AI Usage WebView2 Portable

Claude と Codex の使用量を、デスクトップに常に表示しておける Windows アプリです。
「あとどれくらい使えるか」をひと目で確認できます。

![AI Usage WebView2 Portable screenshot](docs/screenshot.png)

## できること

- Claude と Codex の残量を **横並びで常時表示**
- **5時間枠**（短期）と**週間枠**の残量を、パーセンテージとバーで表示
- 残量が減ると **バーと数字の色が自動で変化**（白→黄→赤）
- リセットまでの残り時間や日時を表示
- 使い切ったときは **カード全体が赤く変化** して一目でわかる
- 最前面に固定できるので、作業中もちらっと確認できる
- ウィンドウはドラッグで移動、右下をドラッグでサイズ変更

## 使い方

### 起動

`AiUsageWebView2.exe` をダブルクリックするだけです。

### 初回ログイン

初めて使うときは、Claude と Codex にログインする必要があります。
画面に「ログイン」ボタンが表示されるので、それをクリックしてブラウザ上でログインしてください。
一度ログインすれば、次回以降は自動でデータが取得されます。

### 画面の見方

```
┌──────────────────────────────────────────────────────┐
│  ● Codex          ⚡ ↻   ● Claude          ⚡ ↻    │
│  5時間 残り 72%  ████████░░       5時間 使用 45%  ████░░░░░░    │
│                リセットまで 3h42m                 リセットまで 2h10m │
│  週   残り 69%  ██████░░░░       週   使用 30%  ███░░░░░░░    │
│                リセット 5/24 9:49                リセット 5/18 1:00 │
└──────────────────────────────────────────────────────┘
```

- **●** サービスの識別マーク（Claude: 青、Codex: 緑）
- **⚡** ブーストボタン（高頻度更新に切り替え）
- **↻** 手動更新ボタン
- **残り / 使用** 設定で切り替えられます
- **バーの色** 残量50%以下で黄色、30%以下で赤に変化
- **右端のアイコン** ×（閉じる）、📌（最前面固定）、⚙（設定）
- 各ボタンにマウスを合わせるとツールチップが表示されます

### 自動更新

使用量は自動で更新されます。手動で更新したいときは、各カード右上の更新ボタン（↻）をクリックしてください。F5キーでも全体を更新できます。

| 状況 | 更新間隔 |
|------|---------|
| 通常時 | 5分ごと |
| ブースト ON のとき | 1分ごと（30分間） |
| リセットが近いとき（15分以内） | 自動で1分ごとに切り替わる |

**ブースト**: 「今すぐ頻繁に確認したい」ときに、カード右上のトグルをONにすると、30分間だけ1分間隔で更新されます。

### 表示の状態

| 見た目 | 意味 |
|--------|------|
| 通常のカード | 正常にデータ取得中 |
| 「古い」バッジ | しばらくデータが更新されていない |
| カード全体が赤い + 「上限」バッジ | 使い切った状態。リセット時刻も表示される |

## 設定

右端の ⚙ アイコンから設定画面を開けます。設定を変えると、すぐにメイン画面に反映されます。

### 変更できる項目

| 項目 | 説明 | 初期値 |
|------|------|--------|
| 通常更新間隔 | 普段の更新頻度 | 5分 |
| ブースト時間 | ブースト ON の継続時間 | 30分 |
| ブースト更新間隔 | ブースト中の更新頻度 | 1分 |
| リセット直前の自動切替 | リセット何分前から頻繁に更新するか | 15分前 |
| リセット直前の更新間隔 | 自動切替後の更新頻度 | 1分 |
| 表示モード | 残量表示 / 使用量表示 | 残量表示 |
| 黄色になる残量 | バーが黄色に変わるしきい値 | 50% |
| 赤になる残量 | バーが赤に変わるしきい値 | 30% |
| 文字サイズ | ラベル・パーセント・リセット時刻 | — |
| 言語 | 日本語 / English | 日本語 |
| 最前面に固定 | 他のウィンドウの上に常に表示 | ON |

## 別のPCで使いたい

フォルダごとコピーするだけです。インストール不要、レジストリも使いません。

必要なファイル:
- `AiUsageWebView2.exe`
- 同じフォルダ内の `.dll` ファイルすべて

コピー先のPCに Microsoft Edge が入っていれば動きます（Windows 10以降なら普通は入っています）。

## うまく動かないとき

- **「ログインが必要」と表示される** → ログインボタンを押して、ブラウザ画面でログインしてください
- **数値が更新されない** → 更新ボタン（↻）をクリックするか、F5を押してみてください
- **それでもダメなとき** → `%USERPROFILE%\.ai-usage-widget\` フォルダにログが保存されています。`claude-webview2.txt` や `codex-webview2.txt` を開くと、取得されたデータの中身を確認できます

## ビルド

Visual Studio は不要です。以下を実行すると `bin\AiUsageWebView2.exe` が生成されます。

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\build.ps1
```

初回は NuGet から WebView2 パッケージを自動ダウンロードします。

## 仕組み

裏側で Edge の埋め込みブラウザ（WebView2）を非表示で2つ起動し、Claude と Codex の使用量ページにアクセスしてテキストを取得しています。取得したテキストから正規表現で使用率やリセット時刻を抽出し、自前のウィンドウに描画しています。

ログイン状態は `%LOCALAPPDATA%\AiUsageWebView2\WebView2Profile` に保存されます。
